## Development Setup

1. **Fork and Clone**
2. **Install Dependencies**: `pnpm install`
3. **Build in Watch Mode**: `pnpm build --watch`
4. **Link for Testing**: 
   - In `vpi` folder: `pnpm link --global`
   - In your test docs: `pnpm link --global vitepress-ai-i18n`

## Pull Request Process

1. Ensure your code passes linting.
2. Update the README.md if you are adding new features.
3. The maintainers will review your PR as soon as possible.
